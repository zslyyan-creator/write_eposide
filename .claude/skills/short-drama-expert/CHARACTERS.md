# 人物设计深度指南

## 人物设计核心原则

**角色存在的唯一理由**：推动剧情或制造冲突

没有这个作用的角色，删掉。

**极致化原则**：角色要"发疯"——坏要坏到丧心病狂，惨要惨到惨绝人寰，通过极致的情绪表达快速击中观众。

---

## 人设极致化设计

### 为什么要极致化？

短剧时长有限，没有时间让观众慢慢了解角色。必须用**最极端的方式**让观众在几秒内记住角色、爱上角色或恨透角色。

### 极致化公式

```
普通人设 × 10倍放大 = 短剧人设
```

### 各类角色的极致化设计

| 角色类型 | 普通设定 | 极致化设定 |
|---|---|---|
| **女主（惨）** | 被欺负 | 被当众扇耳光、跪在雨中、所有人背叛 |
| **女主（强）** | 有能力 | 一句话让对手破产、一个眼神让全场噤声 |
| **男主（宠）** | 对女主好 | 为她屠城、为她放弃江山、眼里只有她 |
| **男主（狠）** | 有手段 | 笑着让仇人生不如死、温柔地说出最狠的话 |
| **反派（恶）** | 使坏 | 害死女主全家、抢走女主一切、还要踩着她上位 |
| **反派（贱）** | 心机 | 当面姐姐妹妹、背后捅刀子、还装无辜 |

### 极致化示例

**普通版恶婆婆**：
> 不喜欢儿媳妇，经常挑刺

**极致版恶婆婆**：
> 当众扇儿媳耳光、逼她跪着伺候、把她赶出家门还要让她净身出户、联合小三陷害她、甚至要害她的命

**普通版霸总**：
> 有钱有势，喜欢女主

**极致版霸总**：
> 全城首富、一个电话让女主仇人公司倒闭、为了女主和全家族对抗、受伤流血还要先确认女主安全、平时冷面阎王只对女主一人露出笑容

## 主角设计四维模型

### 第一维：外在形象

**功能**：让观众快速记住角色

| 元素 | 作用 | 示例 |
|---|---|---|
| 外貌特征 | 视觉记忆点 | 一头红发、左脸疤痕、总穿白衬衫 |
| 标志性动作 | 行为记忆点 | 紧张时转戒指、生气时咬下唇 |
| 口头禅 | 语言记忆点 | "有意思"、"你确定？"、"无所谓" |
| 职业形象 | 身份标签 | 白大褂=医生、西装=精英、围裙=厨师 |

**设计技巧**：
- 外在形象要与内在性格形成呼应或反差
- 反差更出彩：看起来凶实际温柔、看起来弱实际强大

### 第二维：社会身份

**功能**：提供冲突的土壤

| 元素 | 冲突来源 |
|---|---|
| 职业 | 职场竞争、专业对决 |
| 阶层 | 门第之见、阶级跨越 |
| 家庭关系 | 婆媳矛盾、兄弟阋墙 |
| 社会关系 | 朋友背叛、敌人联手 |

**设计技巧**：
- 主角的社会身份要能产生天然冲突
- 示例：穷女孩嫁入豪门（阶层冲突）、女强人遇到大男子主义老公（观念冲突）

### 第三维：内在缺陷

**功能**：角色弧光的起点

**缺陷类型**：

| 类型 | 表现 | 弧光方向 |
|---|---|---|
| 性格缺陷 | 自卑、骄傲、懦弱、冲动 | 克服缺陷，变得更好 |
| 认知盲区 | 太信任人、看不清真相 | 认清现实，变得清醒 |
| 心理创伤 | 童年阴影、过去伤痛 | 治愈创伤，获得新生 |
| 坚守的肥皂泡 | 错误的信念、虚假的希望 | 泡泡破碎，接受真相 |

**设计技巧**：
- 缺陷必须在剧情中被利用或被克服
- 缺陷要让观众既心疼又着急
- 示例：女主太善良总被利用 → 观众急得想冲进屏幕帮她

### 第四维：核心欲望

**功能**：驱动角色行动

| 层次 | 定义 | 示例 |
|---|---|---|
| 外部目标（Want） | 角色明确想要的东西 | 复仇、夺回公司、证明清白 |
| 内心需求（Need） | 角色真正缺失的东西 | 被爱、被认可、找到自我 |

**设计技巧**：
- Want 和 Need 要形成张力
- 示例：女主想要复仇（Want），但真正需要的是放下过去获得幸福（Need）
- 故事结尾：获得 Want 的同时满足 Need = 完美结局

## 对手设计原则

### 对手的核心功能

1. **阻挡主角**：让主角的目标难以实现
2. **逼迫成长**：让主角不得不变强
3. **映射主角**：对手是主角的镜像或反面

### 对手设计三要素

**1. 合理动机**
```
❌ 错误：反派就是坏，没有理由
✅ 正确：反派有自己的立场和理由

示例：
- 恶婆婆不是天生坏，是因为儿子是她唯一的依靠
- 情敌不是无缘无故抢男人，是因为她也爱了很多年
- 商业对手不是纯粹使坏，是因为利益冲突
```

**2. 足够威胁**
```
对手必须能真正威胁到主角，否则没有紧张感

威胁层级：
- 初级对手：能给主角制造麻烦
- 中级对手：能让主角遭受重大损失
- 终极对手：能让主角陷入绝境
```

**3. 镜像关系**
```
对手往往是主角的另一种可能

示例：
- 主角善良 ↔ 对手狠辣（如果主角不坚守会变成对手）
- 主角隐忍 ↔ 对手张扬（两种处世方式的对比）
- 主角重情 ↔ 对手重利（价值观的对立）
```

### 反派层级设计

| 层级 | 功能 | 出场时机 | 结局 |
|---|---|---|---|
| 小喽啰 | 制造日常冲突 | 全程 | 随时被打脸 |
| 假对手 | 前期主要冲突 | 第一二幕 | 中期被解决 |
| 真对手 | 核心冲突 | 第三幕揭露 | 结尾被击败 |
| 幕后黑手 | 终极Boss | 最后揭露 | 大结局对决 |

## 配角设计原则

### 配角的功能分类

| 类型 | 功能 | 示例 |
|---|---|---|
| 助攻型 | 帮助主角、提供信息 | 闺蜜、兄弟、导师 |
| 阻碍型 | 制造麻烦、增加难度 | 恶婆婆、势利亲戚 |
| 映衬型 | 对比主角、突出特点 | 软弱的姐妹、失败的前任 |
| 功能型 | 推动剧情、交代信息 | 律师、医生、路人 |

### 配角设计要点

1. **不要抢戏**：配角再出彩也不能盖过主角
2. **要有记忆点**：一个特征足够，不需要太复杂
3. **服务主线**：每个配角都要对主线有作用
4. **可以工具化**：功能型配角不需要深度刻画

## 人物关系设计

### 关系网构建

```
                    [终极对手]
                        ↑
                      对抗
                        ↑
[助攻配角] ←—帮助—→ [主角] ←—对抗—→ [假对手]
                        ↑
                      感情
                        ↓
                    [感情线对象]
                        ↑
                      竞争
                        ↓
                    [情敌]
```

### 关系张力设计

| 关系类型 | 张力来源 | 示例 |
|---|---|---|
| 爱情 | 误会、阻碍、竞争 | 男女主之间的推拉 |
| 友情 | 背叛、考验、牺牲 | 闺蜜反目或生死与共 |
| 亲情 | 偏心、误解、和解 | 重男轻女的父母 |
| 敌对 | 利益、仇恨、立场 | 商业对手、情敌 |

## 人物设计自检清单

### 主角检查
- [ ] 有明确的外部目标吗？
- [ ] 有需要克服的缺陷吗？
- [ ] 观众能代入/共情吗？
- [ ] 有足够的行动力吗？（不能太被动）
- [ ] 弧光设计清晰吗？

### 对手检查
- [ ] 动机合理吗？
- [ ] 能真正威胁主角吗？
- [ ] 和主角有镜像关系吗？
- [ ] 有让观众恨的点吗？

### 配角检查
- [ ] 对主线有作用吗？
- [ ] 有记忆点吗？
- [ ] 会不会抢主角的戏？

### 整体检查
- [ ] 人物关系有张力吗？
- [ ] 每个角色都不可替代吗？
- [ ] 人物数量会不会太多？（短剧建议核心人物不超过6个）
