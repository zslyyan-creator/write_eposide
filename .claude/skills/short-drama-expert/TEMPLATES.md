# 统一模板格式定义

本文件定义了短剧创作过程中所有输出的标准 Markdown 格式。**所有输出必须严格遵循这些模板**。

---

## 一、项目信息模板

**文件名**：`00_项目信息.md`

```markdown
# 《项目标题》

## 基本信息

| 字段 | 内容 |
|---|---|
| **标题** | 《XXX》 |
| **类型** | 甜宠/复仇/逆袭/穿越/战神/赘婿 |
| **集数** | XX集 |
| **单集时长** | 1-2分钟 |
| **目标受众** | 女性向/男性向/全年龄 |
| **创建日期** | YYYY-MM-DD |

## 一句话卖点

> [身份反差] + [核心困境] + [关键行动] + [悬念钩子]

示例：
> 被渣男当众退婚的她，转身嫁给了全城最神秘的男人。所有人等着看笑话，却不知她的新婚丈夫，是让渣男跪着叫爸爸的存在……

## 核心爽点

- **主爽点**：打脸/逆袭/身份反转/宠溺/碾压...
- **副爽点**：...

## 标题验证

- [ ] 有"网感"（反差/悬念/身份标签/数字）
- [ ] 能引发好奇心
- [ ] 暗示核心冲突
- [ ] 目标受众会点击

## 备注

（其他需要记录的信息）
```

## 格式规范速查

### 符号系统

| 符号 | 用途 | 示例 |
|---|---|---|
| `△` | 动作/神态描写 | △林青宴冷笑，放下茶杯。 |
| `**人名**` | 台词前人名加粗 | **林青宴**（冷笑）：哦？是吗。 |
| `（语气）` | 表情/动作提示 | （咬牙）、（故作镇定）、（压低声音） |
| `OS` | 内心独白 | **林青宴**OS：这笔账我记下了。 |
| `VO` | 画外音 | **小欢**VO：老大，目标到了。 |
| `【闪回】` | 回忆开始 | 【闪回】 |
| `【闪回结束】` | 回忆结束 | 【闪回结束】 |
