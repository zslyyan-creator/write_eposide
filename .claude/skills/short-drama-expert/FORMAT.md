# 短剧剧本格式规范

## 场次编号系统

```
集数-场次
```

| 示例 | 含义 |
|---|---|
| 1-1 | 第1集第1场 |
| 1-2 | 第1集第2场 |
| 2-1 | 第2集第1场 |
| 10-3 | 第10集第3场 |

**需要更换场次的情况**：
1. 空间发生变化
2. 同一空间但时间不连续（间隔较长）

---

## 场景头格式

```
场次编号 景：具体地点，时间，空间
人物：出场人物（若干路人）
```

**时间**：日 / 夜
**空间**：内 / 外

**示例**：
```
1-1 景：顾家客厅，日，内
人物：林青宴、柳眉、顾灿灿
```

---

## 符号系统

### △ 动作/神态描写

用于描写演员能演出来的动作、神态、表情。

```
△林青宴驻足在落地窗前，指尖微微颤抖。
△顾燕之缓缓睁开眼，眼中满是质疑。
△张艳的脸色瞬间变得难看。
```

**要点**：
- 必须是可拍摄的视觉内容
- 简洁有力，不要冗长
- 突出关键动作和微表情

---

### 台词格式

**人名加粗 + 语气提示用括号**：

```
**柳眉**（不耐烦）：放肆！主人家没说话，你就直接入座，没有教养。

**林青宴**（冷笑）：这不就是恶毒婆婆的经典台词吗？
```

**语气提示类型**：
- 情绪类：愤怒、惊讶、冷笑、得意、心虚、急切
- 动作类：起身、转头、压低声音、咬牙
- 状态类：醉酒、虚弱、强撑、故作镇定

---

### OS 内心独白

角色心里想但没说出来的话。

```
**林青宴**OS（惊疑）：十倍？妈妈的生命是无价的！

**顾燕之**OS：这个女人，到底是什么来头？
```

**使用场景**：
- 揭示角色真实想法与表面言行的反差
- 交代观众需要知道但角色不会说出的信息
- 制造悬念或讽刺效果

---

### VO 画外音

人物声音出现但人不在画面中。

```
**小欢**VO（急促）：老大，目标已经到达指定位置。

**廖淑瑶**VO（电话那头）：妈，你能给我转三千万吗？
```

**常见场景**：
- 电话通话
- 画外旁白
- 回忆中的声音

---

### 【闪回】回忆片段

```
【闪回】
1-3 景：医院走廊，夜，内
人物：林青宴、林华

△三年前。林华躺在病床上，握着林青宴的手。

**林华**（虚弱）：宴宴，妈妈对不起你……

【闪回结束】
```

**注意**：
- 闪回需要单独编场次
- 明确标注【闪回】和【闪回结束】
- 闪回内容要简短，服务于当前剧情

---

## 完整场景示例

```
1-1 景：顾家客厅，日，内
人物：林青宴、柳眉、顾灿灿

△顾家别墅装修奢华，水晶吊灯璀璨夺目。
△柳眉端坐在沙发正中央，手里把玩着一串佛珠。
△顾灿灿百无聊赖地刷着手机。
△林青宴款款走入，环顾四周。

**柳眉**（眼皮都不抬）：来了？

△林青宴直接在旁边的单人沙发上坐下。

**柳眉**（冷哼）：放肆！主人家没说话，你就直接入座，没有教养。

△林青宴不慌不忙，自己倒了杯茶。

**顾灿灿**（嫌弃）：连最起码的礼仪都不知道，不知道哪里来的野女人。

△林青宴端起茶杯，嗤笑出声。

**林青宴**（挑眉）：既然你们看我不顺眼，怎么不把廖淑瑶叫回来？

△顾灿灿气得一把摔碎林青宴的茶杯。

**林青宴**OS：呵，这才刚开始，好戏还在后头呢。

—— 本集完 ——
```

## 单集结构模板

```
第X集

1-1 景：XXX，日/夜，内/外
人物：XXX、XXX
[字幕：如有需要]

△环境描写（一句话）
△人物动作/状态

**角色A**（语气）：台词。

△动作反应

**角色B**（语气）：台词。

**角色A**OS：内心独白。

---

1-2 景：XXX，日/夜，内/外
人物：XXX、XXX

[继续...]

---

【本集卡点】
△关键画面定格描写

—— 本集完 ——
```

---

## 格式自检清单

- [ ] 场次编号是否正确（集数-场次）
- [ ] 景/时/空间/人物是否标注完整
- [ ] △符号是否用于所有动作描写
- [ ] 人名是否加粗
- [ ] 语气提示是否用括号
- [ ] OS/VO是否使用正确
- [ ] 闪回是否有开始和结束标记
- [ ] 每场是否有明确的冲突或推进
- [ ] 结尾卡点是否设计到位
