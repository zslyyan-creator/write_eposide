# 产品思维与用户洞察

## 核心理念：短剧是商品，不是艺术品

> 编剧应将短剧视为"互联网时代能高效变现的商品"，站在观众立场提供"服务"。

**产品经理思维**：
- 创作前必须调研用户画像
- 挖掘用户自己都未察觉的心理诉求
- 目标是激发观众的**冲动消费**

---

## 一、三大商业化衡量标准

| 标准 | 定义 | 检验问题 |
|---|---|---|
| **对抗（Conflict）** | 激烈且全面的冲突 | 主角与谁对抗？与环境对抗？与自我对抗？ |
| **变化（Change）** | 强反转、强成长、出人意料 | 主角从A到B的变化够大吗？观众能预料到结局吗？ |
| **共情（Empathy）** | 观众在角色中找到自己的倒影 | 观众会代入主角吗？会为主角的遭遇愤怒/心疼吗？ |

**自检公式**：
```
爆款潜力 = 对抗强度 × 变化幅度 × 共情深度
```

## 二、三层爽点设计体系

### 第一层：动物级爽点（底层驱动）

**本能层面的刺激**，直击人类最原始的欲望：

| 类型 | 触发点 | 示例 |
|---|---|---|
| **生存** | 绝境求生、逆袭翻盘 | 从被扫地出门到身价百亿 |
| **暴力** | 复仇、碾压、征服 | 打脸羞辱过自己的人 |
| **两性** | 占有、被宠、争夺 | 霸道总裁独宠、多人追求 |

**设计要点**：这是最底层的驱动力，必须有，但不能只有这个。

### 第二层：人性级爽点（心理归属）

**满足深层心理需求**：

| 类型 | 心理机制 | 示例 |
|---|---|---|
| **优越感** | 通过主角打脸反派获得心理优势 | 曾经看不起我的人跪着求我 |
| **秩序回归** | 将混乱整理为有序的成就感 | 夺回遗产、查明真相、沉冤昭雪 |
| **公平正义** | 坏人得到报应的满足感 | 渣男被绿、恶婆婆破产 |
| **被认可** | 从不被看好到万众瞩目 | 隐藏身份揭露后全员跪舔 |

### 第三层：情绪拉扯（高黏性秘诀）

**不仅要"爽"，还要通过"虐"来拉扯情绪**：

```
爽 → 虐 → 更爽 → 更虐 → 超级爽
 ↑                              ↓
 └──────── 情绪过山车 ←─────────┘
```

**拉扯公式**：
```
黏性 = 爽感强度 × 虐感深度 × 交替频率
```

**示例节奏**：
```
第1-3集：女主被虐到极致（虐）
第4-5集：女主小反击成功（爽）
第6-8集：反派更狠的报复（更虐）
第9-10集：女主身份初露（更爽）
第11-15集：女主被设计陷入绝境（超虐）
第16-20集：男主出手+女主觉醒（超级爽）
```

## 三、用户画像与心理诉求

### 女性向用户核心诉求

| 诉求 | 对应剧情 | 爽点设计 |
|---|---|---|
| **被无条件宠爱** | 霸总独宠、只对她温柔 | 反差萌、吃醋、护短 |
| **逆袭打脸** | 从灰姑娘到女王 | 曾经看不起她的人后悔 |
| **被争抢** | 多个优质男追求 | 男二痴情、男主吃醋 |
| **复仇快感** | 惩罚渣男贱女 | 以牙还牙、加倍奉还 |

### 男性向用户核心诉求

| 诉求 | 对应剧情 | 爽点设计 |
|---|---|---|
| **身份碾压** | 扮猪吃虎、隐藏大佬 | 打脸狗眼看人低的人 |
| **实力至上** | 一拳解决问题 | 武力/财力/权力碾压 |
| **美女环绕** | 后宫、红颜知己 | 多个美女主动靠近 |
| **翻盘逆袭** | 从废物到巅峰 | 曾经的敌人跪求原谅 |

## 四、题材选择与微创新

### 经典题材"旧瓶装新酒"

**可参考的经典框架**：
- 20年前韩剧：虐恋、财阀、灰姑娘
- 港片经典：公平正义、身份反转、兄弟情义
- 网文爆款：重生、穿越、系统、金手指

### 微创新技巧（低风险高回报）

**原则**：在验证过的框架内进行小幅改动

| 原设定 | 微创新 | 新鲜感来源 |
|---|---|---|
| 战神归来 | 假战神（其实是替身） | 身份悬念加倍 |
| 重生复仇 | 从穿越后第5年开始讲 | 跳过铺垫直接高能 |
| 霸总甜宠 | 霸总是社恐/恋爱脑 | 反差萌 |
| 赘婿逆袭 | 赘婿是女装大佬 | 性别反转 |
| 真假千金 | 真千金主动让位 | 反套路 |

### 热点借势

**灵感来源**：
- 微博热搜榜
- 抖音热门BGM评论区
- 小红书情感话题
- 知乎热门问答

**热梗植入**：
- "王妈"式反套路台词
- 网络流行语自然融入
- 社会情绪共鸣点

## 五、产品化创作检查清单

### 创作前检查

- [ ] 目标用户是谁？（性别/年龄/观看场景）
- [ ] 用户的核心心理诉求是什么？
- [ ] 这个题材有验证过的成功案例吗？
- [ ] 我的微创新点是什么？
- [ ] 对抗/变化/共情三要素都有吗？

### 创作中检查

- [ ] 三层爽点都设计了吗？（动物级/人性级/情绪拉扯）
- [ ] 爽和虐的节奏交替了吗？
- [ ] 观众能在主角身上找到自己吗？
- [ ] 每个反派都让观众恨得牙痒痒吗？
- [ ] 每个爽点都让观众拍手叫好吗？

### 创作后检查

- [ ] 如果我是观众，我会付费吗？
- [ ] 付费点的情绪积累足够吗？
- [ ] 有没有让观众"出戏"的地方？
- [ ] 整体节奏是否一直在线？

## 六、爆款公式总结

```
爆款短剧 = 精准用户洞察
         × 三层爽点设计
         × 情绪过山车节奏
         × 经典框架微创新
         × 极致产品化执行
```

**记住**：你不是在创作艺术品，你是在调制一瓶让人"上头"的功能性饮料。

- 精准研究口渴的人想喝什么（产品思维）
- 加入大量糖分和咖啡因（极致爽点与情绪）
- 确保每一口都有新味道（快节奏与卡点）
- 换一个时髦的包装（微创新与视觉质感）
- 让人一喝就上头，忍不住买下一瓶
