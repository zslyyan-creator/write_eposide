# 卡点设计与付费点策略

## 什么是卡点

卡点是短剧的核心商业设计——在观众情绪最强烈的时刻切断，驱动付费或继续观看。

**卡点的本质**：制造"未完成感"，利用蔡格尼克效应（人们对未完成事项的记忆更深刻）。

---

## 三大卡点类型

### 1. 悬念卡点

**原理**：关键信息即将揭露时切断

**适用场景**：
- 身份即将暴露
- 真相即将揭晓
- 关键人物即将出场
- 重要决定即将宣布

**示例**：
```
场景：女主被逼问新婚丈夫的身份

恶婆婆
你嫁的那个穷小子叫什么？我倒要看看是哪个不要脸的敢娶你！

女主微微一笑，不慌不忙。

女主
他叫——

【画面定格，下集预告闪过男主霸气侧脸】

—— 本集完 ——
```

**设计要点**：
- 问题必须是观众也想知道的
- 答案必须有冲击力
- 切断时机要精准（话说一半）

---

### 2. 情绪卡点

**原理**：在情绪最高点或最低点切断

**适用场景**：
- 主角被羞辱到极致（愤怒）
- 主角即将反击（期待）
- 感情线高甜时刻（心动）
- 误会最深时刻（揪心）

**示例（最惨时刻）**：
```
场景：女主被所有人背叛

女主跪在雨中，看着曾经的闺蜜挽着自己的未婚夫，所有人都在嘲笑她。

闺蜜
（得意）
林小姐，不，应该叫林乞丐了。你以为你还是千金大小姐吗？

众人哄笑。

女主缓缓抬头，眼中的泪水混着雨水，但嘴角却勾起一抹笑。

女主
（低声）
很好，我记住了。

【画面定格在女主那抹诡异的笑容上】

—— 本集完 ——
```

**示例（最爽时刻）**：
```
场景：男主身份即将碾压全场

众人还在嘲笑女主嫁了个穷光蛋。

这时，一队黑衣保镖鱼贯而入，所有人都愣住了。

领头保镖走到男主面前，恭敬鞠躬。

保镖
顾总，车已经备好。

全场死寂。

恶婆婆的脸色瞬间惨白。

恶婆婆
（颤抖）
顾……顾总？！

男主淡淡扫了她一眼。

男主
怎么，现在知道怕了？

【画面定格在恶婆婆惊恐的脸上】

—— 本集完 ——
```

**设计要点**：
- 情绪要铺垫到位，不能突然爆发
- 切在情绪顶点，不要切在下坡
- 给观众一个"解气"或"揪心"的理由继续看

---

### 3. 反转卡点

**原理**：刚出现意外转折时切断

**适用场景**：
- 以为赢了其实输了
- 以为是敌人其实是盟友
- 以为死了其实活着
- 以为真相其实是假象

**示例**：
```
场景：女主以为扳倒了对手

女主
（得意）
证据确凿，你还有什么话说？

反派女二脸色惨白，似乎已经认输。

就在这时，门被推开。

一个女主从未见过的男人走了进来，他看了女主一眼，嘴角带着玩味的笑。

神秘男人
有意思，林小姐，你以为这就结束了？

女主愣住。

神秘男人
（转向反派女二）
别怕，有我在。

反派女二瞬间恢复了笑容。

反派女二
姐姐，好戏才刚刚开始呢。

女主的脸色变了。

【画面定格】

—— 本集完 ——
```

**设计要点**：
- 反转要出乎意料但合乎情理
- 新信息要足够颠覆之前的认知
- 切在"新局面刚刚展开"的时刻

## 付费点布局策略

### 付费点位置（以80集为例）

| 位置 | 集数 | 剧情节点 | 卡点类型 |
|---|---|---|---|
| 第一付费点 | 8-12集 | 激励事件后/第一次反击前 | 情绪卡点（期待） |
| 第二付费点 | 25-30集 | 中点反转 | 反转卡点 |
| 第三付费点 | 45-50集 | 一切尽失/最低谷 | 情绪卡点（揪心） |
| 第四付费点 | 65-70集 | 最终对决前 | 悬念卡点 |

### 付费点设计原则

1. **前置爽点**：付费点前必须有足够的爽感铺垫，让观众"上头"
2. **悬念最大化**：付费点的悬念要比普通卡点更强
3. **不可跳过**：付费点的信息必须是理解后续剧情的关键
4. **情绪峰值**：付费点要在情绪曲线的峰值或谷值

## 每集卡点节奏

### 标准单集结构（1.5分钟）

```
0:00-0:03  开场钩子（承接上集悬念或新冲突）
0:03-0:30  冲突建立
0:30-1:00  冲突升级
1:00-1:20  小高潮/反转
1:20-1:30  卡点
```

### 卡点前的铺垫技巧

**错误示范**：
```
女主突然发现了真相。
—— 本集完 ——
```
（没有铺垫，观众没有期待感）

**正确示范**：
```
女主盯着那份文件，手开始颤抖。

女主
（喃喃）
不可能……这不可能……

她猛地抬头，看向一直信任的闺蜜。

女主
（声音发抖）
这份文件……是你给他的？

闺蜜的笑容僵在脸上。

—— 本集完 ——
```
（有情绪铺垫，观众和女主一起震惊）

## 卡点禁忌

| ❌ 禁止 | 原因 |
|---|---|
| 在无聊的对话中切断 | 观众没有继续看的欲望 |
| 连续多集同类型卡点 | 审美疲劳 |
| 卡点后下集不兑现 | 观众感觉被骗 |
| 卡点信息无关紧要 | 浪费观众期待 |
| 强行制造悬念 | 逻辑不通会让观众出戏 |

---

## 卡点自检清单

每个卡点设计完成后，检查：

- [ ] 观众会想知道接下来发生什么吗？
- [ ] 这个悬念/情绪是自然铺垫出来的吗？
- [ ] 下一集能否很好地承接这个卡点？
- [ ] 这个卡点比前几集的卡点更强吗？（递进原则）
- [ ] 切断的时机是最佳时机吗？（不早不晚）
