# 短剧爆款剧本创作专家 (Short Drama Expert)

本项目是用于创作爆款短剧剧本的专业技能文档集合，核心目标是创作出具备高商业价值和高付费转化率的竖屏短剧。

## 核心创作理念

本技能基于短剧商业化成功经验总结出以下核心理念：

### 爆款铁三角
- **对抗** + **变化** + **共情**

### 商业化铁律
1. 黄金3秒必须有钩子（Hooks）
2. 每90秒一个小高潮
3. 每集结尾必有卡点（Cliffhanger）
4. 付费点前必须有足够的情绪积累

## 五步创作流程

本技能严格遵循以下五步流程，每一步完成后都需要用户确认才能进入下一步，确保剧本质量和商业化方向。

1. **标题与核心创意**: 确定项目定位、类型、卖点、受众，并写入项目信息文件。
2. **故事简介与梗概**: 创作投流简介和四幕结构梗概。
3. **人物小传**: 为主角、对手、配角创建详细的角色卡。
4. **分集大纲**: 逐集拆解剧情，设计卡点，并进行付费点验证。
5. **剧本正文**: 按照规范的剧本格式逐集撰写可拍摄的正文。
