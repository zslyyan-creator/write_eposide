# 短剧爆款剧本创作专家 (Short Drama Expert)

本项目是用于创作爆款短剧剧本的专业技能文档集合，核心目标是创作出具备高商业价值和高付费转化率的竖屏短剧。

> **核心定位**：短剧是商品，不是艺术品。你是在调制一瓶让人"上头"的功能性饮料。

## 核心创作理念

### 爆款铁三角
- **对抗（Conflict）**：激烈且全面的冲突
- **变化（Change）**：强反转、强成长、出人意料
- **共情（Empathy）**：让观众在角色中找到自己的倒影

### 上瘾三要素
1. **永远不让心理闭环完成** - 解决一个问题，立刻抛出新问题
2. **高频情绪奖励** - 每30秒一个情绪波动点
3. **递进式悬念设计** - 悬念强度不断升级，信息分批释放

### 三层爽点体系
1. **动物级爽点**：生存、暴力、两性（底层驱动）
2. **人性级爽点**：优越感、秩序回归、公平正义（心理归属）
3. **情绪拉扯**：爽→虐→更爽→更虐→超级爽（高黏性秘诀）

### 商业化铁律
- 黄金3秒必须有钩子
- 每90秒一个小高潮
- 每集结尾必有卡点（毒性等级★★★以上）
- 付费点前必须有足够的情绪积累
- 悬念接力：解决30%旧悬念 → 制造100%新悬念 → 暗示更大悬念
- 人设极致化：坏要坏到丧心病狂，惨要惨到惨绝人寰

## 五步创作流程

本技能严格遵循以下五步流程，每一步完成后都需要用户确认才能进入下一步：

| 步骤 | 内容 | 核心检查点 |
|:---:|:---|:---|
| 1 | **标题与核心创意** | 产品思维验证、三层爽点、微创新点 |
| 2 | **故事简介与梗概** | 四幕结构、情绪曲线、伏笔设计 |
| 3 | **人物小传** | 人设极致化、Want vs Need、角色弧光 |
| 4 | **分集大纲** | 卡点毒性等级、悬念接力、付费点验证 |
| 5 | **剧本正文** | 对白口语化、情绪波动点、上瘾检查 |

## 支持的短剧类型

| 类型 | 核心爽点 | 经典套路 |
|:---|:---|:---|
| 甜宠 | 双向奔赴、反差宠溺 | 契约婚姻、先婚后爱、追妻火葬场 |
| 复仇 | 打脸、逆袭、真相大白 | 重生复仇、隐藏身份、真假千金 |
| 战神 | 身份碾压、实力打脸 | 退役归来、隐藏大佬、神医归来 |
| 赘婿 | 扮猪吃虎、身份反转 | 隐忍三年、一朝觉醒、全员跪舔 |
| 穿越 | 金手指、改变命运 | 带系统、穿书、重生逆袭 |
| 虐恋 | 错过、误会、破镜重圆 | 双向救赎、先虐后甜 |

## 核心公式速查

```
爆款潜力 = 对抗强度 × 变化幅度 × 共情深度

黏性 = 爽感强度 × 虐感深度 × 交替频率

卡点毒性 = 悬念强度 × 情绪积累 × 信息关键度

上瘾循环 = 悬念制造 → 情绪积累 → 即时奖励 → 新悬念 → 循环
```

---

**记住**：好的短剧不是让观众"看完"，而是让观众"停不下来"。
